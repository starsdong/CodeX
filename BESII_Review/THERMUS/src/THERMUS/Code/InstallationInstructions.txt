
 INSTALLATION INSTRUCTIONS
 -------------------------

 Requirements: 1) Users must have ROOT installed (see http://root.cern.ch/)
               2) Users must have a copy of "Numerical Recipes in C" code

  * Unzip and extract the THERMUS source from the zipped tar file;

  * Set an environment variable `THERMUS' to point at the top-level directory containing the THERMUS code;

  * Copy the following "Numerical Recipes in C" functions to \$(THERMUS)/nrc:

		broydn.c  	rsolv.c
		fdjac.c  	fmin.c
		lnsrch.c  	nrutil.c
		nrutil.h  	qrdcmp.c
		qrupdt.c  	rotate.c
		zbrent.c 

  * Use the makefiles in \$(THERMUS)/functions, \$(THERMUS)/nrc and \$(THERMUS)/main to 
    build the libFunctions.so, libNRCFunctions.so and libTHERMUS.so shared 
    object files 

    (i.e. run 'make all' in each of these directories)


  * Finally, in the \$(THERMUS) directory, open a ROOT session, 
    load the libraries and begin

    i.e. inside ROOT:

	root [ ] gSystem->Load("./lib/libFunctions.so");
	root [ ] gSystem->Load("./lib/libNRCFunctions.so");
	root [ ] gSystem->Load("./lib/libTHERMUS.so");

 RUNNING SAMPLE MACROS
 ---------------------

  * In SampleMacro.C and SampleFit.C the paths to the particle properties file and the
    directory containing the particle decay files must be edited to
    match the user's directory structure. In addition, in both SampleMacro.C and
    SampleFit.C the full path for the ExpData.txt file must also be inserted.

  * Then, having loaded the libraries, load the first macro SampleMacro.C and run
    it:

	root [ ] .L SampleMacro.C
        root [ ] SampleMacro()

  * Compare the output to that in SampleMacroOutput.txt

  * Next load SampleFit.C and run it:

	root [ ] .L SampleFit.C
        root [ ] SampleFit()

  * Compare the last "page" of the output to that in SampleFitOutput.txt
