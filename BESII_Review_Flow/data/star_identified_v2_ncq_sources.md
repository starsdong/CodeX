# STAR Identified-Particle v2 NCQ Dataset

Generated by:

```bash
python3 scripts/build_star_identified_v2_ncq.py
```

Outputs:

- `data/star_identified_v2_pt.csv`
- `data/star_identified_v2_ncq.csv`
- `figures/star_identified_v2_ncq/star_identified_v2_ncq_*GeV.{png,pdf}`
- `figures/star_identified_v2_ncq/star_identified_v2_ncq_all_energies.{png,pdf}`

## Sources Used

- 3.0, 3.2, 3.5, 3.9, and 4.5 GeV: STAR, Phys. Rev. Lett. 135, 072301 (2025), HEPData record `10.17182/hepdata.159489`.
  The Fig. 2 tables provide `v2(pT)` for pi+, pi-, K+, K-, K0S, p, and Lambda in 10-40% centrality; the script computes `(mT-m0)/nq` and `v2/nq`.

- 54.4 GeV: STAR, Phys. Lett. B 827, 137003 (2022), HEPData record `10.17182/hepdata.110656`.
  These tables are already in NCQ-scaled form, so `pT` and `v2` in `star_identified_v2_pt.csv` are reconstructed from `(mT-m0)/nq`, `v2/nq`, masses, and nq.

- 7.7, 11.5, 14.5, 19.6, 27, 39, and 62.4 GeV: STAR, Phys. Rev. C 93, 014907 (2016), STAR public 0-80% C++ data table.
  The public table provides `v2(pT)`, and the script computes `(mT-m0)/nq` and `v2/nq`.

- 200 GeV: STAR, Phys. Rev. C 77, 054901 (2008), STAR public Figure 7 data table.
  The public table provides `v2(pT)`, and the script computes the NCQ-scaled coordinates.

## Notes

- Centrality is not uniform across all energies: 3.0-4.5 and 54.4 GeV are 10-40%, while the BES-I plus 200 GeV points are 0-80%.
- Systematic uncertainties from asymmetric public tables are symmetrized with the larger of the low/high uncertainties before plotting.
- The older HEPData records `10.17182/hepdata.95903` and `10.17182/hepdata.102408` are kept locally as fallbacks for 4.5 GeV and BES-I energies, but the current output uses the later STAR PRL low-energy HEPData and the STAR BES-I public table.
